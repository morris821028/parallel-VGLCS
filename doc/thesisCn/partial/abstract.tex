\Abstract{可變的間隙限制最長同子序列應用於基因、分子生物學中。在之前的研究中，已提出理論在 $O(nm)$ 的算法，其使用特殊的 incremental tree set union 完成 $O(1)$ 的操作，否則必須使用 $O(\alpha(n))$ 的并查集操作。原先的序列算法無法以直觀的方式平行，我們修改了原本算法的資料結構，並且提出平行版本的遞增任意區間最大值詢問，平行的運行時間為 $O(nm / p + n \log n)$。在單一處理器上，理論的時間複雜度仍可以達到 $O(nm)$，平行效能勝過於先前的研究。

\noindent \textbf{Keywords.} range minimum query, incremental range maximum query, incremental suffix maximum query, longest common sequence, parallel, cartesian tree
